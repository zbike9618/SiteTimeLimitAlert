# サイト閲覧時間制限アラート (Site Time Limit Alert)

指定したウェブサイトの閲覧時間を制限し、使いすぎを防止するためのGoogle Chrome拡張機能です。
特定のサイト（SNSや動画サイトなど）や、登録済みサイト全体の合計利用時間を管理できます。

## 🚀 主な機能

*   **個別サイト制限**: ドメインごとに1日の閲覧制限時間（分単位）を設定できます。
*   **全体制限 (Global Limit)**: 登録したすべてのサイトの合計閲覧時間に対する制限を設定できます。
*   **強制ブロック**: 制限時間を超えると、その日の閲覧がブロックされ、専用の制限画面が表示されます。
*   **スヌーズ機能**: 制限画面から「あと5分だけ延長」することが可能です。
*   **リアルタイム通知**: アイコンバッジに残り時間が表示され、刻々とカウントダウンされます。
*   **週間統計**: ポップアップ内で過去7日間の閲覧時間をグラフで確認できます。
*   **簡単追加**: 閲覧中のページ上で右クリックし、「制限リストに追加」を選ぶだけで登録可能です。

## 📦 インストール方法

この拡張機能はストア公開前の開発版（野良拡張機能）としての導入を想定しています。

1.  このリポジトリ（フォルダ）をローカルに保存します。
2.  Google Chromeを開き、URLバーに `chrome://extensions/` と入力します。
3.  右上の「デベロッパーモード」を **ON** にします。
4.  左上の「パッケージ化されていない拡張機能を読み込む」をクリックします。
5.  この拡張機能のフォルダ（`manifest.json` が入っているフォルダ）を選択します。

## 🛠️ 使い方

1.  Chromeのツールバーからアイコンをクリックしてポップアップを開きます。
2.  **サイトを追加**: 制限したいドメイン（例: `youtube.com`）と時間（分）を入力して「追加・更新」を押します。または、サイト閲覧中に右クリックメニューから追加も可能です。
3.  **全体制限の設定**: 「全体制限を設定」にチェックを入れると、全サイトの合計使用時間に対するリミットを設定できます。
4.  設定した時間を超えると、画面がブロックされます。
